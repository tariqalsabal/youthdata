# لوحة إحصاءات المستخدمين وتسجيل الدخول — تركيب في Oracle APEX

صفحة APEX واحدة بأسلوب HTML خام + AJAX تعرض **24 مؤشراً رقمياً + 17 مخططاً = 41 إحصائية**
من جدولَي `USERS` و `USER_SESSIONS` (سكيمة `DATA_CENTER`).

## خطوات التركيب (5 دقائق)

### 1) أنشئ صفحة جديدة
App Builder → التطبيق → **Create Page → Blank Page** → الاسم: «لوحة الإحصاءات».
(يُفضّل جعلها تتطلّب تسجيل دخول/للمشرفين فقط.)

### 2) أضف منطقة المحتوى (الواجهة)
- في Page Designer: أضف **Region** نوعه **Static Content**.
- العنوان: اتركه فارغاً أو «الإحصاءات».
- في خاصية **Source → HTML Code**: الصق **كامل محتوى** ملف `DASHBOARD_REGION.html`.
- تأكّد أن **Escape special characters = Off** (حتى تُنفَّذ الأكواد).

### 3) أضف معالج البيانات (AJAX)
- في Page Designer → تبويب **Processing** → انقر يميناً على **Ajax Callbacks → Create Process**.
- **الاسم (مهم جداً):** `GET_DASHBOARD` (مطابق تماماً — الواجهة تناديه بهذا الاسم).
- النوع: **PL/SQL Code**.
- الكود: الصق **كامل محتوى** ملف `DASHBOARD_AJAX_PROCESS.sql`.

### 4) شغّل الصفحة
احفظ ثم **Run**. ستُحمّل البطاقات والمخططات تلقائياً، وزر «تحديث» يعيد الجلب.

## ملاحظات

- **مكتبة الرسوم:** تُحمّل Chart.js من CDN تلقائياً. إن كانت شبكة الخادم تمنع CDN،
  نزّل `chart.umd.min.js` وارفعه في **Shared Components → Static Application Files**،
  ثم بدّل قيمة `CANVAS_LIB` في الـ HTML إلى `#APP_FILES#chart.umd.min.js`.
- **الأرقام العربية:** تظهر بالأرقام العربية الهندية. لأرقام إنجليزية، بدّل في الـ HTML
  `toLocaleString("ar-EG")` إلى `toLocaleString("en-US")`.
- **الأمان:** المعالج للقراءة فقط (SELECT)، ولا يستقبل مدخلات من المستخدم، فلا مخاطر حقن.
- المصدر: التسجيلات من `USERS."تاريخ_التسجيل"`، وتسجيلات الدخول من `USER_SESSIONS.CREATED_AT`.
